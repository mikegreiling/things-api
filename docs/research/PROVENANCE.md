# Provenance

These documents were produced across Codex (GPT 5.x) research sessions in March 2026, originally under `second-brain/docs/ai-gtd-things/`, and recovered/imported into this repository on 2026-07-02 at project bootstrap. They are preserved verbatim as the validated evidence base for this project.

**Treat as source of truth:** `03-automation-capability-matrix.md` (URL-scheme capability matrix v1) and `validation-notes-step3.md` (raw validation observations T01–T20, including DB field findings such as `startBucket`/`todayIndex` and the repeating-item `when`-write crash).

**Context, not current scope:** documents 02, 04–07 concern GTD-workflow operationalization and the Things/Obsidian boundary — explicitly out of scope for the things-api MVP, retained for the follow-on project.

**Superseded assumptions:** these docs deliberately excluded AppleScript and Shortcuts as write vectors. That decision was reversed at project kickoff (2026-07-02); see `../design/lab.md` for the probe program covering both, plus the newly discovered `thingscli` binary and private sdef surfaces.

The capability matrix v2 produced by the VM probe program (`docs/design/lab.md`) will supersede matrix v1; matrix v1 remains the baseline the U-probe suite must reproduce.
